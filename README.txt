Olá, seja bem-vindo ao projeto "Friday-IA". 
Friday é uma assistente virtual que eu estou desenvolvendo, que funciona com o GPT-3 da openai.

Friday está sendo desenvolvida como uma ajudante pessoal para diversos tipos de serviços. Nós procuramos diminuir o tempo gasto tarefas desnecessárias como: passar horas criando listas, pesquisando algum assunto ou resumo entre outras coisas.
